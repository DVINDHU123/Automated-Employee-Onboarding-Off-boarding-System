# 🚀 Automated Employee Onboarding & Offboarding System

## 📌 Project Overview
This project is developed on the **ServiceNow Developer Platform** to automate employee onboarding and offboarding processes. It replaces manual workflows with a structured, catalog-driven system that improves efficiency, transparency, and compliance.

The system allows HR, IT, Facilities, and Security teams to collaborate through automated task assignments and SLA-based tracking.

---

## 🎯 Objectives
- Automate onboarding and offboarding workflows  
- Reduce manual effort and errors  
- Ensure timely task completion using SLAs  
- Provide real-time visibility for HR  
- Improve operational efficiency and compliance  

---

## ⚙️ Features

### 🧑‍💼 Onboarding
- Service Catalog request for new employees  
- Automatic task generation for:
  - IT (Laptop, Email, Access)
  - Facilities (Workspace)
  - Security (ID Card, Access)  
- SLA tracking and notifications  

### 🚪 Offboarding
- Exit request submission  
- Asset return tracking  
- Access revocation  
- Clearance workflow  

### 📊 Dashboard & Reports
- Real-time monitoring  
- Task status tracking  
- SLA performance insights  

---

## 🛠️ Technologies Used
- ServiceNow Developer Instance  
- Service Catalog  
- Flow Designer / Workflow  
- SLA Management  
- Reports & Dashboards  

---

## 🏗️ Workflow
1. User submits request via Service Catalog  
2. Flow/Workflow is triggered  
3. Tasks are assigned to departments  
4. SLA monitors deadlines  
5. Progress is tracked in dashboards
